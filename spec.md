# Idé → Byggklar Spec
## Projekt: Myrenblom.work wow-portfolio
Datum: 2026-02-11

## 1) Mål
Bygga en snygg, konverterande portfolio med få men starka demos inom automation/AI agents. Målet är att vinna konsultförtroende hos CEOs/founders i Stockholm.

## 2) Vem är användaren?
- Primär: potentiella kunder (CEO/founder/ops lead)
- Sekundär: rekryterare/samarbetspartners

## 3) Scope (MVP)
- Hero med tydligt värdeerbjudande
- 3 starka case demos (problem → lösning → mätbar ROI)
- Tjänstepaket + CTA för kontakt/bokning
- Social proof/credibility block
- Enkel, snabb, mobiloptimerad design

## 4) Input/Output
- Input: Leos projektresultat + ROI-exempel
- Output: Körbar portfolio-sajt (static) + README
- Kanal: lokal preview + deploybar struktur

## 5) Constraints
- Fokus på tydlighet före komplexitet
- Snabb initial MVP, sedan iterera
- Undvik fluff, visa konkreta outcomes

## 6) Plan mode
- Arkitektur: single-page responsive site med sektioner
- Steg: content skeleton → visual system → cases → CTA → polish
- Verifiering: Lighthouse/UX-check + copy review

## 7) Definition of Done
- Sidan är live-runnable lokalt
- 3 case med mätbar effekt finns
- CTA och kontaktväg är tydlig
